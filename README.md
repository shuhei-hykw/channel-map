# detector-mapping
Detector-FEE Mapping API

## Requirements

- CMake 3.10

## Installation

The way to install is as follows.

```sh
$ cmake -S . -B .build
$ cmake --build .build
$ cmake --install .build
```

The way to clean is as follows.

```sh
$ cmake --build .build --target clean
```
or
```sh
$ rm -rf .build
```
